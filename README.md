# Task-Manager_App
A Python-based Task Manager Application that lets users add, view, update, and delete tasks. Built using OOP concepts,
# Features
•Add new tasks with name, description, and priority,
•View all saved tasks in a formatted list,
•Update or delete existing tasks,
•Data stored persistently in data/tasks.json,
•Built using OOP principles and JSON file handling,
•Developed during CertifyNXT Internship under Zoblik International,
